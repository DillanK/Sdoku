### 기능 요구사항
- 홈 화면에 쉬움/보통/어려움/극악 4단계 난이도 선택 버튼을 표시한다
- 난이도 선택 시 해당 난이도로 게임 플레이 화면으로 전환한다
- 홈 화면에 난이도별 완료 횟수를 표시한다
- 홈 화면에 난이도별 최고 기록(최단 소요시간)을 표시한다
- 게임 완료 시 난이도, 소요시간, 완료여부를 CoreData에 저장한다
- 미완료 상태로 종료 시에도 게임 기록(완료여부=false)을 CoreData에 저장한다
- 홈 화면 통계는 앱 재시작 후에도 유지된다

### 비기능 요구사항
- CoreData 저장은 메인 스레드를 블로킹하지 않아야 한다 (background context 사용)
- 홈 화면 초기 로딩 시 통계 집계 쿼리는 200ms 이내에 완료되어야 한다
- 네트워크 연결 없이 완전히 오프라인으로 동작해야 한다
- iOS 16 이상 지원

### 인수 기준
- 홈 화면 진입 시 4개의 난이도 버튼이 모두 표시된다
- 각 난이도 버튼 탭 시 해당 난이도의 게임 플레이 화면으로 이동한다
- 게임 완료 후 홈 화면으로 돌아오면 해당 난이도의 완료 횟수가 1 증가해 있다
- 게임 완료 후 홈 화면으로 돌아오면 최고 기록이 갱신된 경우 새 기록이 반영된다
- 앱을 완전히 종료 후 재시작해도 저장된 통계가 그대로 표시된다
- 기록이 없는 난이도는 완료 횟수 0, 최고 기록 없음으로 표시된다
