## Spec 핵심 결정
- 요약: 난이도 선택 UI, CoreData 기반 게임 기록 저장, 홈 화면 통계 요약을 포함한 Sdoku 앱의 진입점 화면 및 데이터 영속성 레이어
- 핵심 요구사항:
  - 홈 화면에 쉬움/보통/어려움/극악 4단계 난이도 선택 버튼을 표시한다
  - 난이도 선택 시 해당 난이도로 게임 플레이 화면으로 전환한다
  - 홈 화면에 난이도별 완료 횟수를 표시한다
  - 홈 화면에 난이도별 최고 기록(최단 소요시간)을 표시한다
  - 게임 완료 시 난이도, 소요시간, 완료여부를 CoreData에 저장한다
- 수용 기준:
  - 홈 화면 진입 시 4개의 난이도 버튼이 모두 표시된다
  - 각 난이도 버튼 탭 시 해당 난이도의 게임 플레이 화면으로 이동한다
  - 게임 완료 후 홈 화면으로 돌아오면 해당 난이도의 완료 횟수가 1 증가해 있다
- 제외 범위: iCloud 동기화 또는 백업, 게임 기록 상세 목록/히스토리 화면, 게임 기록 삭제 기능, GameCenter 연동, 푸시 알림 또는 로컬 알림, 난이도별 통계 이외의 추가 지표(평균 시간, 오답 횟수 등)
- 영향 범위: Sdoku/HomeView.swift (신규), Sdoku/GameRecord.swift (CoreData Entity 신규), Sdoku/PersistenceController.swift (신규 또는 CoreData stack 설정), Sdoku/GameRecordRepository.swift (신규, CRUD 서비스), Sdoku/HomeViewModel.swift 또는 AppState (통계 상태 관리), Sdoku.xcdatamodeld (CoreData 모델 신규), 게임 플레이 화면 → 홈 화면 네비게이션 연결 지점
